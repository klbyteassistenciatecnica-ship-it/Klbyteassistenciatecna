KLByte - Site estático

Arquivos:
- index.html
- style.css
- script.js
- images/klbyte-logo.png
- images/foto1.jpg
- images/foto2.jpg
- images/foto3.jpg
- images/foto4.jpg

Como usar:
1. Faça o download e extraia o ZIP.
2. Substitua as imagens na pasta 'images' pelas suas imagens reais (mesmos nomes de arquivo).
3. Abra index.html em seu navegador para ver o site localmente.
4. Para publicar online, envie a pasta para Netlify, GitHub Pages ou outro serviço de hospedagem.

Observações:
- Atualize contatos em index.html (telefone, email e endereço).
- Se quiser zip com diferente estrutura ou adicionar favicon, me avise.
